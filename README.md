# HSE Calendar Parser

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Python](https://img.shields.io/badge/python-3.11+-blue.svg)](pyproject.toml)

Парсер расписания ФГН НИУ ВШЭ (Нижний Новгород) из Excel (.xlsx) в iCalendar (.ics).

Преобразует сложное многострочное расписание из Google Sheets в чистый календарь, совместимый с Google Calendar, Apple Calendar, Яндекс Календарь и Outlook.

## Возможности

- 📅 Парсинг Excel-файлов с расписанием ВШЭ (1–4 курс)
- 🎯 Фильтрация по группе и подгруппе
- 📍 Автоматическое определение аудитории и корпуса
- 🔄 Обработка отмен, восстановлений и переносов занятий
- 📆 Поддержка недельной чередовки (верхняя/нижняя неделя)
- 🌐 Online-занятия с пометкой `Online`
- 🚫 Пропуск английского языка, физкультуры и MINOR (настраивается)
- 📊 Подробный отчёт о результатах парсинга

## Использование

### 🧙 Интерактивный мастер (рекомендуется)

```bash
python -m hse_schedule_parser
```

Интерактивный помощник проведёт через 5 шагов:
1. **Файл** — укажи путь к `.xlsx` файлу
2. **Группа** — выбери курс и группу (автоопределение из файла)
3. **Подгруппа** — опциональный фильтр по подгруппе
4. **Настройки** — что пропускать (MINOR, английский, физра)
5. **Сохранение** — сохрани `.ics` файл или посмотри отчёт

### ⚡ Командная строка (для продвинутых)

```bash
python -m hse_parser.cli --file <путь-к-xlsx> --group <код-группы> --output <путь-к-ics>
```

#### Примеры

**1 курс, группа 25ФПЛ1, 3 модуль:**

```bash
python -m hse_parser.cli \
  --file "3 модуль.xlsx" \
  --group 25ФПЛ1 \
  --output schedule.ics
```

**С фильтром по подгруппе:**

```bash
python -m hse_parser.cli \
  --file "3 модуль.xlsx" \
  --group 25ФПЛ1 \
  --subgroup 1 \
  --output schedule_subgroup1.ics
```

#### Все аргументы CLI

| Аргумент | Обязательный | Описание |
|----------|-------------|----------|
| `--file`, `-f` | Да | Путь к файлу Excel (.xlsx) с расписанием |
| `--group`, `-g` | Да | Код группы (например, `25ФПЛ1`, `24ФИЛ1`) |
| `--output`, `-o` | Нет | Путь для сохранения файла .ics (по умолчанию: `<group>_module<N>.ics`) |
| `--subgroup` | Нет | Фильтр по подгруппе (`1` или `2`) |
| `--academic-year-start` | Нет | Начало учебного года (по умолчанию: `2025`) |
| `--skip-minor` | Нет | Пропускать MINOR (по умолчанию: `true`) |
| `--skip-english` | Нет | Пропускать английский язык (по умолчанию: `true`) |
| `--skip-pe` | Нет | Пропускать физкультуру (по умолчанию: `true`) |
| `--verbose`, `-v` | Нет | Подробный вывод событий в консоль |

## Импорт календаря

Сгенерированный файл `.ics` можно импортировать в любой календарь:

- **Google Календарь**: Настройки → Импорт → Выбрать файл
- **Apple Календарь**: Файл → Импортировать
- **Яндекс Календарь**: Настройки → Импорт календаря
- **Outlook**: Файл → Открыть и экспортировать → Импорт/экспорт

## Как это работает

1. **Чтение файла**: Парсер открывает Excel-файл и находит колонку с расписанием указанной группы
2. **Парсинг ячеек**: Содержимое каждой ячейки разбивается на блоки (дисциплины), извлекаются даты, тип занятия, название дисциплины и преподаватель
3. **Генерация дат**: Для периодов (например, "с 12.01 по 23.03") генерируются конкретные даты с учётом чётности недель
4. **Экспорт**: Все события сохраняются в формате iCalendar (.ics) с часовым поясом Europe/Moscow

## Коды групп

Первые две цифры кода группы определяют курс:

| Код | Курс |
|-----|------|
| `25...` | 1 курс |
| `24...` | 2 курс |
| `23...` | 3 курс |
| `22...` | 4 курс |

## Архитектура

Проект состоит из двух пакетов:

### `hse_parser/` — ядро парсинга (стабильное)

```
hse_parser/
├── cli.py              # CLI интерфейс (click)
├── config.py           # Pydantic-модель конфигурации
├── controller.py       # Оркестратор пайплайна
├── reader.py           # Чтение Excel (openpyxl)
├── parser/
│   ├── cell_parser.py  # Основной парсер ячеек
│   ├── date_parser.py  # Извлечение дат из текста
│   ├── text_cleaner.py # Нормализация текста
│   └── rules.py        # Правила пропуска ячеек
├── date_engine.py      # Генерация дат из периодов
├── models.py           # Внутренние модели данных
├── exporter.py         # Экспорт в iCalendar
└── utils.py            # Вспомогательные функции
```

### `hse_schedule_parser/` — TUI-интерфейс (v2.1)

```
hse_schedule_parser/
├── __init__.py         # Инициализация пакета
├── __main__.py         # Точка входа: python -m hse_schedule_parser
├── tui.py              # Rich UI компоненты (панели, прогресс, таблицы)
├── wizard.py           # 5-шаговый мастер с состоянием
├── autodetect.py       # Автоопределение курсов, групп, подгрупп из Excel
└── presets.py          # Сохранение/загрузка настроек (JSON)
```
