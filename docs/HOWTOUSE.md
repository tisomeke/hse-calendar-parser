# 📖 Как пользоваться HSE Schedule Parser

Пошаговая инструкция для тех, кто впервые видит терминал.

---

## 🎯 Что делает эта программа

Программа берёт **Excel-файл с расписанием** (такой выдают в учебном офисе ВШЭ)
и превращает его в **файл календаря** (`.ics`), который можно открыть
на телефоне, в Google Календаре, на Mac, в Outlook — и все пары
автоматически появятся в календаре.

**Было:** `1_курс_2_модуль.xlsx` — таблица, которую неудобно смотреть
**Стало:** `schedule_25ФПЛ1.ics` — календарь с парами по дням и времени

---

## 🔧 Подготовка к работе

### Шаг 1. Установи Python

Программа работает на **Python 3.11 или новее**.

**Windows:**
1. Открой браузер, перейди на https://www.python.org/downloads/
2. Скачай последнюю версию Python (нажми жёлтую кнопку Download)
3. Запусти скачанный файл
4. **ВАЖНО:** внизу окна установки поставь галочку «Add Python to PATH»
5. Нажми «Install Now»

**Mac:**
1. Открой Терминал (через Поиск → Терминал)
2. Введи команду: `brew install python@3.13`
   - Если нет `brew` — сначала установи: `/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

**Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install python3 python3-pip python3-venv
```

### Шаг 2. Скачай программу

**Вариант А — через Git (рекомендуется):**
```bash
git clone https://github.com/tisomeke/hse-calendar-parser.git
cd hse-calendar-parser
```

**Вариант Б — скачать ZIP:**
1. Перейди на https://github.com/tisomeke/hse-calendar-parser
2. Нажми зелёную кнопку «Code» → «Download ZIP»
3. Распакуй ZIP в любую папку
4. Открой Терминал и перейди в эту папку:
   ```bash
   cd /путь/к/распакованной/папке/hse-calendar-parser
   ```

### Шаг 3. Создай виртуальное окружение (рекомендуется)

Это нужно, чтобы не засорять систему.

**Windows:**
```bash
python -m venv .venv
.venv\Scripts\activate
```

**Mac / Linux:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

После активации в начале строки терминала появится `(.venv)`.

### Шаг 4. Установи программу

Всё ещё в терминале, выполни:

```bash
pip install -e .
```

Эта команда установит программу и все нужные зависимости.

## 🚀 Запуск программы

### Простой способ — мастер (TUI)

Просто напиши в терминале:

```bash
python -m hse_schedule_parser
```

Откроется **интерактивный мастер**, который проведёт тебя по шагам.
Управление: стрелки `↑↓` для выбора, `Enter` для подтверждения,
`Space` для переключения галочек.

### Быстрый способ — командная строка (CLI)

Если ты уже знаешь параметры, можно одной командой:

```bash
python -m hse_parser.cli --file "sheet.xlsx" --group "25XXX" --output "my.ics"
```

## 📋 Пошаговая работа с мастером

На каждом шаге доступны кнопки:
- `← Назад` — вернуться к предыдущему шагу
- `✕ Выход` — выйти из мастера

### Шаг 1. Выбор файла

При запуске мастер **автоматически сканирует текущую папку** в поиске `.xlsx` файлов.

**Если файл найден:**
```
Найден файл: 1_курс_2_модуль.xlsx
1 — продолжить с найденным файлом
2 — указать полный путь к таблице с расписанием
```
Выбери `1`, чтобы использовать найденный файл, или `2`, чтобы указать другой.

**Если файл не найден:**
```
В этой директории файл не найден.
Перемести файл с таблицей в эту директорию или укажи полный путь.
```
Введи полный путь к файлу, например:
- `C:\Users\Имя\Desktop\1_курс_2_модуль.xlsx` (Windows)
- `/Users/Имя/Desktop/1_курс_2_модуль.xlsx` (Mac)

**Где взять этот файл?** Файлы расписания выкладывают в учебный офис
или на странице своей образовательной программы на сайте ВШЭ.
Обычно название вроде `1_курс_2_модуль.xlsx` или `2_курс_3_модуль.xlsx`.

### Шаг 2. Выбор курса

Мастер покажет список обнаруженных курсов: `1 курс`, `2 курс` и т.д.

Выбери свой курс стрелками `↑↓` и нажми `Enter`.

### Шаг 3. Выбор группы

Мастер покажет список групп для выбранного курса.

Найди свою группу (например, `25ФПЛ1`, `24БИВ2` и т.д.) и выбери её.

### Шаг 4. Фильтр по подгруппе (если нужно)

Если в расписании есть разделение на подгруппы, мастер спросит:
- «Все подгруппы» — будут все пары
- «Подгруппа 1» — только пары для первой подгруппы
- «Подгруппа 2» — только пары для второй подгруппы

Если подгрупп нет — этот шаг пропустится автоматически.

### Шаг 5. Настройки

Мастер покажет список того, что можно **пропустить** (исключить из календаря):

- `[x] Пропускать MINOR` — убрать дисциплины MINOR
- `[x] Пропускать английский язык` — убрать английский
- `[x] Пропускать физкультуру` — убрать физру

**Как менять:** нажми `Space` (пробел), чтобы убрать или поставить галочку.
По умолчанию всё отмечено (всё пропускается).

**Учебный год определяется автоматически** — мастер сам понимает,
осень сейчас или весна, и подставляет правильный год.

### Шаг 6. Сохранение

Мастер предложит варианты:
- `1 — сохранить в текущую директорию` — файл `schedule_25ФПЛ1.ics` появится в папке с программой
- `2 — указать другой путь` — введи свой путь для сохранения

### Шаг 7. Импорт календаря

После сохранения `.ics` файла его нужно открыть в приложении «Календарь».

**На iPhone / Mac:**
1. Найди файл в Finder (или на экране «Файлы»)
2. Нажми на него — откроется приложение «Календарь»
3. Нажми «Добавить»

**Google Календарь (с компьютера):**
1. Открой https://calendar.google.com
2. Слева в меню нажми «Настройки» (шестерёнка)
3. Слева выбери «Импорт и экспорт»
4. Нажми «Выберите файл на компьютере» → выбери `.ics` файл
5. Нажми «Импортировать»

**Яндекс Календарь:**
1. Открой https://calendar.yandex.ru
2. Нажми на шестерёнку → «Импорт календаря»
3. Выбери файл и нажми «Импортировать»

**Outlook:**
1. Открой Outlook
2. Файл → Открыть и экспортировать → Импорт/экспорт
3. Выбери «Импорт iCalendar (.ics)»

## 🐛 Если что-то пошло не так

### «Python не найден» / «python: command not found»
**Windows:** переустанови Python и не забудь поставить галочку «Add Python to PATH»
**Mac:** попробуй `python3` вместо `python`
**Linux:** установи Python: `sudo apt install python3`

### «Нет модуля rich» / «ModuleNotFoundError»
Ты забыл установить зависимости. Выполни:
```bash
pip install -e .
```

### «Файл не найден»
Проверь, что файл существует и путь указан правильно.
Можно перетащить файл в окно терминала — путь подставится сам.

### «Не удалось прочитать файл»
Убедись, что файл не повреждён и открывается в Excel.
Попробуй сохранить его заново из Excel (Файл → Сохранить как → .xlsx).

### Ошибка при импорте в Google Календарь
Попробуй открыть `.ics` файл в блокноте — если там иероглифы,
значит файл повреждён. Запусти программу заново.

### Ничего не помогает
Создай issue на GitHub: https://github.com/tisomeke/hse-calendar-parser/issues
Опиши, что ты делал, и приложи скриншот ошибки.
