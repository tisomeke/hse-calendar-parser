# ❓ Часто задаваемые вопросы (FAQ)

## Общие вопросы

### Что делает эта программа?
Преобразует Excel-файл расписания ВШЭ (`.xlsx`) в файл календаря (`.ics`),
который можно импортировать в Google Календарь, Apple Календарь, Outlook
и другие приложения.

### Это официальное приложение ВШЭ?
Нет. Это неофициальный инструмент, созданный студентами для студентов.
ВШЭ не имеет к нему отношения.

### Программа платная?
Нет. Программа бесплатна и распространяется с открытым исходным кодом
по лицензии MIT ([LICENSE](LICENSE)).

## Установка и запуск

### Какая версия Python нужна?
Python 3.11 или новее.

### Как проверить, установлен ли Python?
Открой терминал и выполни:
```bash
python --version
```
Если видишь `Python 3.11.x` или выше — всё в порядке.
Если ошибка — установи Python с python.org.

### На Windows команда `python` не работает
Попробуй `py` или `py -3`. Если не помогло — переустанови Python
и при установке поставь галочку «Add Python to PATH».

### На Mac команда `python` не работает
Используй `python3` вместо `python`.

### `pip install -e .` выдаёт ошибку
Убедись, что ты находишься в папке с программой (где лежит `pyproject.toml`).
Проверь командой `ls` (Mac/Linux) или `dir` (Windows) — должен быть
файл `pyproject.toml` в списке.

### Что такое «виртуальное окружение»?
Это изолированная среда для Python-программ. Позволяет не засорять
систему лишними пакетами. Создаётся командой `python -m venv .venv`,
активируется — `.venv\Scripts\activate` (Windows) или
`source .venv/bin/activate` (Mac/Linux).

## Работа с программой

### Где взять Excel-файл с расписанием?
Файлы расписания выкладывают в учебном офисе или на странице
образовательной программы на сайте ВШЭ. Обычно название вроде
`1_курс_2_модуль.xlsx` или `2_курс_3_модуль.xlsx`.

### Программа не видит мою группу
Убедись, что выбран правильный курс. Если группа всё равно не
появляется — возможно, файл имеет нестандартный формат. Создай
issue на GitHub.

### Что такое подгруппа?
Некоторые пары проводятся отдельно для части группы (например,
лабораторные работы). В расписании это обозначается как «гр.1» и «гр.2».

### Зачем пропускать MINOR, английский, физру?
По умолчанию эти дисциплины исключаются из календаря, потому что
они часто нерегулярны или не нужны в основном календаре.
Если они нужны — сними галочки в настройках.

## Формат и импорт

### Что такое .ics?
Стандартный формат файлов календаря (iCalendar). Поддерживается
всеми современными календарями.

### Как открыть .ics на iPhone?
Найди файл в приложении «Файлы» и нажми на него — откроется
«Календарь» с предложением добавить события.

### Как открыть .ics в Google Календаре?
Настройки (шестерёнка) → Импорт и экспорт → выбери файл →
Импортировать.

### Как открыть .ics в Яндекс Календаре?
Настройки → Импорт календаря → выбери файл → Импортировать.

### Как открыть .ics в Outlook?
Файл → Открыть и экспортировать → Импорт/экспорт →
Импорт iCalendar (.ics).

### Можно ли импортировать .ics на телефоне Android?
Да. Открой файл через приложение «Файлы» — система предложит
открыть в «Календаре». Или загрузи файл в Google Календарь
через браузер.

### События в календаре не обновляются при изменении расписания
Нужно скачать новый `.xlsx` файл, запустить программу заново
и импортировать новый `.ics`. Старый календарь можно удалить и старые импортированные события удалить из вашего приложения-календаря вручную.

## Проблемы и ошибки

### «ModuleNotFoundError: No module named 'rich'»
Не установлены зависимости. Выполни:
```bash
pip install -e .
```

### «Файл не найден» при правильном пути
На Windows используй обратную косую черту `\` или двойную `\\`.
Можно перетащить файл в окно терминала — путь подставится сам.

### Программа зависла / не реагирует
Нажми `Ctrl+C` (на Mac — `Cmd+C`), чтобы выйти. Запусти заново.

### Ошибка «Не удалось прочитать файл»
Файл может быть повреждён или иметь нестандартный формат.
Попробуй открыть его в Excel и сохранить заново (Файл → Сохранить как).

### В .ics файле иероглифы / кракозябры
Файл повреждён. Запусти программу заново. Если проблема повторяется —
создай issue на GitHub.

### При импорте в Google Календарь ошибка
Попробуй открыть `.ics` в текстовом редакторе — файл должен
начинаться с `BEGIN:VCALENDAR`. Если нет — перезапусти программу.
